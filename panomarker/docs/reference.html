Google Maps API Addons Reference

PanoMarker class

This class extends google.maps.OverlayView.

Constructor

Constructor Description
PanoMaker(opts?:PanoMarkerOptions) Creates a PanoMarker with the options specified. If a panorama is specified, the marker is added to the map upon construction. Note that the position must be set for the marker to display. Important: do not use the inherited method setMap() to change the panorama, but use setPano() instead, otherwise a proper functionality is not guaranteed.

Events

Events Arguments Description
add HTMLDivElement This event is fired when the marker has been created and drawn on panorama.
click MouseEvent This event is fired when the marker icon was clicked.
remove HTMLDivElement This event is fired when the marker has been destroyed and removed from the panorama.
